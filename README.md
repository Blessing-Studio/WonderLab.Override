## 介绍

WonderLab是一个跨平台 Minecraft 启动器，支持自动更新，Modrinth、Curseforge 资源下载，全格式整合包安装（大概），并支持 Windows，MacOS，Linux 等操作系统，同时兼容 x86,x64,Arm 系统架构。
